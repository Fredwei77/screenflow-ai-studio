# ScreenFlow AI 核心演示路径 QA

目标：验证“在线老师录课”路径能稳定完成。

## 环境

- 前端：`npm.cmd run dev -- --host 127.0.0.1`
- 后端：`npm.cmd run server`
- 浏览器：Chrome 或 Edge
- URL：首页 `http://127.0.0.1:3000/meet/`
- URL：产品工作台 `http://127.0.0.1:3000/meet/workspace`

## 演示路径

1. 打开首页。
2. 点击“开始 AI 录课”。
3. 允许摄像头和麦克风权限。
4. 选择屏幕或双画面。
5. 点击开始录制。
6. 说一段 30-60 秒课程内容。
7. 如需展示互动，打开标注、白板或提词器能力。
8. 点击停止录制。
9. 确认出现视频下载按钮。
10. 确认出现 Pro 升级提示。
11. 点击“申请 Pro”进入意向提示或 pricing。
12. 进入 pricing 后提交 Pro 申请表。

## 验收标准

- 连续 3 次可以完成录制、停止、下载。
- 至少一次能看到字幕或转写内容。
- 录制完成后能看到 Pro 升级提示。
- pricing 页面能提交申请并生成申请编号。
- 产品工作台能看到最新线索或商业意向事件。
